# RCVST-GG-Audio-Disasm
RoboCop vs. The Terminator (Game Gear/Master System) audio disassembly

A disassembly of the sound engine and data from RoboCop vs. The Terminator for Game Gear (also covers the Master System version), fully labeled and commented. Can be built with WLA-DX, provided the file is linked from another "main" source file; the sound data of the resulting ROM is verified to match the sound data from the original ROM exactly. However, the resulting ROM itself on its own is not playable, as no additional code is included. A text file is also included which documents how the music, sound effect, and instrument data is structured for this game as well as other games using versions of the same sound engine.
Note that this disassembly currently does not cover the voice samples, which are handled in a separate routine.
